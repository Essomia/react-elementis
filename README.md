# React Elementis

A development sandbox for custom-made React components.

## Features

- Use `vite` as a bundler.
- Use `eslint` as javascript linter.
- Use `storybook` to develop React component in isolation.

## Installation

1. Clone repository into your machine.
2. Run:
   ```shell
   cd react-elementis
   npm install
   npm run dev
   npm run storybook
   ```
