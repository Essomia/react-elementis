# Welcome to React Elementis

This storybook setup helps you build UI components in isolation from your app's business logic, data, and context.
See it as a development sandbox for custom-made React components.

## Main approaches

- Build UIs with a <a href="https://componentdriven.org" target="_blank" rel="noopener noreferrer"><strong>component-driven</strong></a> process starting with atomic components and ending with pages.
- Use mock data to render pages. This makes it easy to build and review page states without needing to navigate to them in your app.
